斗地主游戏，后端基于Python+Tornado+MySQL开发，前端 Phaser 引擎

**Dependencies**

* Python3.6+
* Mysql5.7+


Quick Start

    git clone 
    cd doudizhu
    mysql --user=root -p < schema.sql
    pip3 install -r requirements.txt
    cd doudizhu
    python3 app.py --password=your_database_password
    Now visit http://127.0.0.1:8080
